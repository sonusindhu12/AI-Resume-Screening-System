AI Resume Screening System

📌 Project Overview

The AI Resume Screening System is a machine learning-based application designed to automate the resume screening process. The system analyzes resumes, extracts relevant information, and classifies candidates into different job categories based on their skills, experience, and qualifications. This helps recruiters save time and improve the efficiency of candidate shortlisting.

🎯 Objectives
Automate the resume screening process.
Extract key information from resumes.
Classify resumes into appropriate job categories.
Reduce manual effort in recruitment.
Improve hiring efficiency using AI and Machine Learning.

📂 Dataset

The project uses a resume dataset containing:

Resume Text
Candidate Skills
Job Category Labels

The dataset is preprocessed to remove noise and prepare the text for machine learning models.

🔄 Project Workflow

Data Collection
Load resume dataset.
Explore dataset structure.
Data Preprocessing
Remove special characters and URLs.
Convert text to lowercase.
Remove stopwords.
Tokenization and text cleaning.
Feature Engineering
Convert text data into numerical format using TF-IDF Vectorization.
Model Building
Train Machine Learning models for resume classification.
Evaluate model performance.
Prediction
Input a new resume.
Predict the most suitable job category.
